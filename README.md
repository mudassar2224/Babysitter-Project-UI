## 📱 Project Overview

This is a beautiful Flutter UI project that provides screens and components for a babysitting application. The app allows users to browse, book, and manage babysitter services through an intuitive mobile interface.

## ✨ Features

- **User Authentication** - Secure login and signup screens
- **Babysitter Profiles** - Browse detailed babysitter information and credentials
- **Service Booking** - Easy booking interface with date and time selection
- **User Dashboard** - Manage bookings and account settings
- **Reviews & Ratings** - View and leave reviews for babysitters
- **Payment Integration UI** - Payment method selection and processing screens
- **Real-time Chat** - Communication interface between users and babysitters
- **Search & Filter** - Find babysitters by location, availability, and ratings

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0 or higher)
- Dart SDK
- Android Studio or Xcode
- A device or emulator for testing

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mudassar2224/Babysitter-Project-UI.git
cd Babysitter-Project-UI/babysitter
```

2. Get dependencies:
```bash
flutter pub get
```

3. Run the app:
```bash
flutter run
```
